HTML (HyperText Markup Language)

HTML adalah bahasa markup standar untuk membuat halaman web. HTML mendeskripsikan struktur halaman web.

Struktur Dasar HTML

<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Judul Halaman</title>
</head>
<body>
  <h1>Ini Heading 1</h1>
  <p>Ini paragraf.</p>
</body>
</html>

Elemen HTML Penting

  • <div> - Container untuk pengelompokan
  • <p> - Paragraf
  • <a href="..."> - Link
  • <img src="..." alt="..."> - Gambar
  • <ul> dan <li> - Daftar tidak berurutan

CSS (Cascading Style Sheets)

CSS adalah bahasa yang digunakan untuk mendesain tampilan dokumen HTML. CSS mengontrol layout, warna, font, dan aspek visual lainnya.

Cara Menambahkan CSS

/* CSS Internal */
<style>
  body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
  }
</style>

/* CSS Eksternal */
<link rel="stylesheet" href="style.css">

/* CSS Inline */
<p style="color: red;">Teks ini merah</p>

Contoh CSS Selector

/* Selektor elemen */
p {
  color: blue;
}

/* Selektor class */
.nama-class {
  font-size: 16px;
}

/* Selektor ID */
#nama-id {
  background-color: yellow;
}

/* Selektor kombinasi */
div p {
  margin: 10px;
}

JavaScript

JavaScript adalah bahasa pemrograman yang memungkinkan Anda membuat konten dinamis di halaman web.

Dasar-dasar JavaScript

// Variabel
let nama = "Budi";
const umur = 25;

// Fungsi
function sapa(nama) {
  return "Halo, " + nama + "!";
}

// Event
document.getElementById("tombol").addEventListener("click", function() {
  alert("Tombol diklik!");
});

// Manipulasi DOM
document.querySelector("h1").textContent = "Judul Baru";

Tipe Data JavaScript

  • String: "teks"
  • Number: 42, 3.14
  • Boolean: true, false
  • Array: [1, 2, 3]
  • Object: { nama: "Budi", umur: 25 }

JSON (JavaScript Object Notation)

JSON adalah format pertukaran data yang ringan dan mudah dibaca oleh manusia dan mesin.

Struktur JSON

{
  "nama": "Budi",
  "umur": 25,
  "pekerjaan": "Developer",
  "hobi": ["membaca", "berkoding", "bermusik"],
  "alamat": {
    "jalan": "Jalan Merdeka No. 123",
    "kota": "Jakarta"
  }
}

Menggunakan JSON di JavaScript

// Mengubah objek ke JSON string
const objek = { nama: "Budi", umur: 25 };
const jsonString = JSON.stringify(objek);

// Mengubah JSON string ke objek
const objekDariJson = JSON.parse(jsonString);

Markdown

Markdown adalah bahasa markup ringan untuk membuat teks yang diformat menggunakan sintaksis yang mudah dibaca dan ditulis.

Sintaksis Dasar Markdown

# Heading 1
## Heading 2
### Heading 3

*Teks miring* atau _Teks miring_
**Tebal** atau __Tebal__

- Daftar tidak berurutan
- Item kedua

1. Daftar berurutan
2. Item kedua

[Link](https://contoh.com)
![Gambar](gambar.jpg)

`Kode inline`

```javascript
// Blok kode
function contoh() {
  return "Halo";
}
```

Contoh Hasil Markdown

Heading 1

Heading 2

Teks miring dan Tebal

  • Daftar tidak berurutan
  • Item kedua
