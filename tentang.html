About YONGCODE

YONGCODE is a minimalist yet powerful web-based code editor designed for developers who want a clean, distraction-free coding environment. Built with modern web technologies, it offers essential features for web development without unnecessary bloat.

Key Features

  • File System Access - Open and save files directly using the File System Access API
  • Folder Navigation - Browse entire project directories with the file explorer
  • Tabbed Interface - Work with multiple files simultaneously
  • Syntax Highlighting - File type detection with appropriate icons
  • Line Numbers - Easy code navigation with synchronized line numbers
  • Customizable - Adjust font size and theme to your preference
  • Run Code - Quickly test HTML/CSS/JS code in a new window

Supported File Types

YONGCODE supports common web development file types including:

.html, .css, .js, .json, .md, and .txt
